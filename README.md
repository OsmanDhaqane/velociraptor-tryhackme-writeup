# Velociraptor TryHackMe Writeup

This repository contains my writeup for the **Velociraptor** room on TryHackMe.

📄 **Writeup (PDF):** [velociraptor-tryhackme-writeup.pdf](velociraptor-tryhackme-writeup.pdf)

In this room, I learned how to use Velociraptor for endpoint monitoring, digital forensics, and incident response. The exercises included deploying an Instant Velociraptor server, connecting a Windows client, collecting forensic artifacts, using VQL (Velociraptor Query Language), exploring endpoint data, and investigating a simulated PrintNightmare attack.

## Skills Covered

- Velociraptor deployment
- Client and server setup
- Endpoint monitoring
- Artifact collection
- VQL (Velociraptor Query Language)
- File system analysis
- Windows Registry access
- PowerShell execution
- PE file analysis
- PrintNightmare detection
- Digital forensics
